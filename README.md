# Mi Closet Smart (React)

Aplicación completa en React para organizar prendas, filtrar por categoría, marcar favoritas y consultar estadísticas básicas del closet.

## Requisitos

- Node.js 18+
- npm 9+

## Ejecutar localmente

```bash
npm install
npm start
```

La app se levanta en `http://localhost:3000`.

## Scripts disponibles

- `npm start`: inicia en modo desarrollo.
- `npm test`: ejecuta pruebas (sin watch).
- `npm run build`: genera build de producción.
